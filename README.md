TaskTwit

* [The Hit List](http://www.potionfactory.com/thehitlist/)
* [Remember The Milk](http://www.rememberthemilk.com/)
* [Producteev](http://www.producteev.com/) [It can sync to [Astrid](http://weloveastrid.com/)]
* Reminders.app (on OS X)
